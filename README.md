# Restaurant Backend API 🍽️

This is a backend API for a Restaurant Management System built using ASP.NET Core.

## Features
- JWT Authentication
- Order Management
- User Login & Registration
- Clean Architecture (API, Application, Infrastructure)

## Tech Stack
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- JWT Authentication

## How to Run
1. Clone the repo
2. Configure connection string in appsettings.json
3. Run the project
